# WTBot
War thunder base bombing bot written with python  
- Completely external
- Open source
- Pretty shitty

It's not malware, I just can't justify spending hundreds of hours to fly cool pixel planes

## Required modules:
CV2, numpy, pillow, time, pyautogui, random, math, pydirectinput

## Usage:
To begin, download or clone the repo and move it somewhere easy to remember.  
`git clone https://github.com/GangstaD69/WTBot.git`

Next, cd into the Launcher folder.  
`cd C:/Users/GangstaD/Downloads/WTBot/Launcher`

Finally, run the launcherui.py file.  
`python launcherui.py`

From there, edit the values in the boxes and ensure all are valid. Once ready, press the run button **once** and tab back into the game.
